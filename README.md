# JavaScript 在线加密平台

这是一个基于本地浏览器的 JavaScript 文件加密工具，支持输入代码或上传 `.js` 文件进行加密，并可自定义加密密钥和版权声明。

演示地址：https://jsjiami.937788.xyz

仓库地址：https://github.com/a937750307/javascript-file-encryption-tool

## 功能

- 输入 JavaScript 代码并加密
- 上传本地 `.js` 文件并自动加载内容
- 自定义加密密钥（建议 32 字符以上）
- 自定义版权声明
- 生成可复制的加密结果
- 测试运行加密后的代码
- 下载加密后的 JavaScript 文件

## 使用说明

1. 打开 `index.html` 文件。
2. 在“输入代码”标签页中粘贴代码，或切换到“上传文件”标签页选择一个 `.js` 文件。
3. 输入加密密钥。
4. 可选：修改版权声明文本。
5. 点击“加密代码”按钮生成加密结果。
6. 使用“复制到剪贴板”、“测试运行加密代码”或“下载加密文件”按钮进行后续操作。

## 文件说明

- `index.html`：主页面和加密逻辑。
- `crypto-js.min.js`：CryptoJS 库，用于加密和哈希计算。
- `atom-one-dark.min.css`、`all.min.css`：样式文件。
